# 페이지 이동<br>
리스트에 페이징 처리를 해 두어 한 페이지에 10개씩 게시글이 나온다.
리스트 아래에 있는 페이지 숫자를 누르면 이동할수 있고 이동한 페이지가 유지된다.<br>
또한 이동할 페이지를 입력 후 이동버튼을 눌러도 동일하게 이동한다.<br><br>
